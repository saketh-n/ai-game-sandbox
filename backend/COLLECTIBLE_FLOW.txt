KEY ALGORITHMS
==============

Connected Component Analysis (Segmentation):
--------------------------------------------
Input: RGBA sprite sheet image
Output: List of individual sprite images

1. Extract alpha channel
2. Create binary mask (transparent vs non-transparent)
3. Label connected regions (scipy.ndimage.label)
4. Find bounding box for each region
5. Filter out noise (< 1% of largest sprite)
6. Sort left-to-right
7. Crop each sprite
8. Convert to base64 data URL


Spawn Position Generation:
--------------------------
Input: List of platform rectangles
Output: List of {x, y, sprite_index} positions

For i in range(num_collectibles):
  attempt = 0
  while attempt < max_attempts:
    1. platform = random.choice(platforms)
    2. x = random(platform.x + margin, platform.x + width - margin)
    3. y = platform.y - 20
    4. if distance to all placed > min_spacing:
         positions.append({x, y, random_sprite_index})
         break
    5. attempt++


Collision Detection (Phaser.js):
--------------------------------
this.physics.add.overlap(group1, group2, callback)

- Runs every frame
- Checks bounding box overlap
- When overlap detected: calls callback
- Callback can destroy/modify sprites
